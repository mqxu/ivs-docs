用户操作手册

一、主要功能/模块操作流程

1. 车辆绑定/解绑

1.1 车辆绑定

  1. 登录后进入“车辆管理”页面
  2. 点击“添加车辆”按钮
  3. 输入车牌号、车辆品牌、VIN 等信息，上传车辆照片
  4. 点击“提交绑定”，等待系统审核/绑定成功提示
  5. 成功后在车辆列表可见新车辆

1.2 车辆解绑

  1. 在“车辆管理”页选择待解绑车辆
  2. 点击“解绑”按钮，确认解绑操作
  3. 系统提示解绑成功，车辆列表自动刷新

2. 充电服务

2.1 扫码充电

  1. 进入“充电地图”页，定位附近充电桩
  2. 点击目标桩点，进入详情页，选择“扫码充电”
  3. 扫描充电桩二维码，核对车辆信息
  4. 点击“启动充电”，可实时查看充电进度与剩余时间
  5. 充电完成后，系统自动推送账单/充电记录

2.2 充电历史与账单查询

  1. 进入“充电记录”或“历史数据”页面
  2. 筛选车辆/日期，查看详细充电历史与账单详情
  3. 支持导出 PDF/Excel 报表

3. 数据查询与可视化

  1. 在“数据大屏”或“历史分析”页面，选择车辆、时间区间
  2. 查看能耗、行驶里程、充电趋势等可视化图表
  3. 点击图表可查看更多细节或下载数据

4. 告警处理

  1. 进入“消息中心”或“告警中心”页面
  2. 新告警以红点/高亮提示
  3. 点击告警消息查看详情与处理建议
  4. 已处理告警自动归档，可随时查询历史记录

二、新用户注册、登录与账户管理

  1. 打开 App 或 Web 端首页，点击“注册/登录”
  2. 输入手机号/邮箱，设置密码或选择微信/第三方快捷登录
  3. 完成实名认证和信息完善后，即可进入系统主页
  4. 个人中心可随时修改昵称、头像、密码
  5. 如忘记密码，点击“找回密码”按流程重置

三、车辆管理与数据可视化操作指南

  1. 车辆管理：可绑定多辆车，支持分组查看、添加/删除、车辆档案编辑
  2. 数据可视化：所有历史数据、趋势分析通过大屏/报表集中展示，支持单车/多车对比分析
  3. 支持一键导出图表数据，便于报销、分析和运维

四、常见问题解答（FAQ）

问题解决方法
忘记密码无法登录？点击“找回密码”，按短信/邮箱提示操作
绑定车辆审核不通过？核对输入信息与车辆照片清晰度，必要时联系客服
充电启动失败？检查车辆与桩点状态，确认网络与权限，必要时重启 App
地图定位不准？确认定位授权已开启，尝试刷新或重启设备
账单与充电记录数据不同步？检查网络，若持续异常联系客服
小程序/Pad 无法扫码？确认摄像头权限，必要时切换至 App 或联系客服

六、移动端与 Web 端操作差异提示

  • UniApp/小程序端适合移动使用，支持扫码、地图导航、推送等原生能力
  • Web 端适合大屏操作与数据导出，部分原生功能（如扫码）在 Web 可能受限
  • 小程序端如遇授权限制可切换 App 或 Web 继续操作
  • 数据、消息、告警等信息多端实时同步

七、故障处理与联系客服方式

  • 自动排查：遇到 App 闪退、卡顿、操作无响应，建议重启 App/浏览器/设备
  • 问题记录：如遇异常，截图报错信息与操作步骤，便于反馈
  • 联系客服：可通过“个人中心-联系客服”入口，或致电 400-xxxx-xxx，或添加官方企业微信/公众号留言
  • 在线帮助：常见问题和教程见“帮助中心”/FAQ 页面，24 小时自动响应
