课程概述

本课程以智能车载系统为核心项目，采用项目化教学模式，融合移动应用开发理论与实践。课程内容涵盖：

🚗 核心功能模块

  • 车辆信息管理: 电池电量(SOC)、里程跟踪、胎压监测
  • 用户车辆绑定: 账户系统、手机/微信登录、VIN 码扫描
  • 智能充电服务: 充电桩查找、实时充电、计费系统
  • 数据分析可视化: 驾驶报告、能耗分析、交互式图表

💻 技术架构

  • 前端: uni-app (跨平台)、Vue3 组件
  • 后端: Spring Boot 3、RESTful APIs
  • 数据库: MySQL 8
  • 可视化: ECharts 数据图表分析
  • 开发工具: HBuilderX、IDEA、Navicat
  • 云服务: 阿里云 ECS/Serverless、华为云、腾讯云

📚 课程结构

课程材料按照软件开发生命周期组织，包含 8 个主要模块：

完整开发流程

从课程设计 → 需求调研 → UI 设计 → 系统架构 → 开发实现 → 测试交付 → 展示归档

快速开始

选择以下任一模块开始学习：

  • 📋 课程教学设计 - 了解课程整体设计和教学安排
  • 🔍 需求与调研 - 学习需求分析和用户研究方法
  • 🎨 原型与 UI 设计 - 掌握界面设计和交互原理
  • 🏗️ 系统架构设计 - 深入理解系统架构和数据库设计
  • ⚡ 接口与开发 - 实现 API 接口和核心功能模块
  • 🧪 测试与交付 - 学习测试方法和部署流程
  • 📊 展示与归档 - 项目展示和成果总结

学习目标

通过本课程学习，学生将能够：

  1. 掌握全栈开发技能 - 从前端到后端的完整开发能力
  2. 具备工程化思维 - 理解软件工程的标准流程和最佳实践
  3. 提升竞赛能力 - 为移动应用开发竞赛做好充分准备
  4. 培养创新思维 - 结合 AI 工具提升开发效率和质量

注意事项

本课程注重实践操作，建议学生准备好开发环境，并积极参与项目实战。
